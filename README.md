# Produit Service

Service de gestion des produits basé sur Spring Boot.

## Démarrage

```bash
./mvnw spring-boot:run
```

## Structure

- `controller/` - API REST
- `service/` - Logique métier
- `repository/` - Accès aux données
- `model/` - Entités

## Configuration

Voir `application.properties` pour les paramètres de l'application.
